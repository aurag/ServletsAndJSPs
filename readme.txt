Présentation du projet :
Application web basique suivant le design pattern MVC2.
Un servlet unique (MVC2Servlet) gère toutes les requêtes de l'utilisateur. 
Il est conçu de manière à être facilement évolutif, des requêtes peuvent facilement être rajoutées.
De la documentation vous aidera à comprendre son fonctionnement et à le maintenir ou le faire évoluer.

Fonctionnalités du projet :
	- Création d'évènements avec un nom, un type, une description, une date de début et une date de fin
	- Edition d'un évènement existant : tous les champs indiqués plus haut sont modifiables 
	- Suppression d'un évènement
	- Persistance des évènements grâce à une base de données
	- Affichage des évènements déjà existants sur la page d'accueil
	- Affichage du détail d'un évènement en particulier pour voir toutes les informations sur celui-ci

Framework utilisés :
	- Bootstrap twitter pour un affichage plus agréable visuellement pour l'utilisateur
	- Jquery pour un format de sélection des dates pour simple et agréable pour l'utilisateur (datePicker)
